[기본 기능 구현]

[구상]

게임 메인 해상도 사이즈 960*600  16:10비율
16:9로 바꿀까 고민중 (11/12)

타일 사이즈 155*230 로 일단 통일함

클라이언트 접속 -> [메인 화면 panel] -> 이름 입력창, 입장,  설정 버튼,
백그라운드 이미지, gif 등

[설정 panel]: 마스터 볼륨, bgm볼륨, 효과음 볼륨 등 설정 기능 아마(400*300)

이름 입력 -> 입장 누르면 -> [로비 panel] 접속

[로비 panel]: 좌상단에 접속중인 클라이언트(닉네임, 클라이언트에게 쪽지 보내기 기능?), 
방 목록(방 이름, 비밀방 여부, 인원 ex) 1/2, 게임중 or 준비중), 방 목록 새로고침 버튼,
하단에 접속한 클라이언트간 자유로운 채팅 text area

방 만들기 버튼 ->[방 만들기 panel] 띄움(400*300)
생성돼있는 방 클릭 후 입장 버튼 클릭 -> [게임 룸 panel]로 입장

[로비 panel]좌상단에 띄워진 접속하고 있는 클라이언트 옆에 있는 쪽지 버튼 클릭시에 ->[쪽지보내기 panel]띄움

[쪽지보내기 panel] : 내용 입력 text area, 전송 버튼, 취소 버튼 (400*300)

[방 만들기 panel]: 방 이름 입력 text area, 비밀번호 설정여부(체크박스)-> 비밀번호 설정 text area(4자리 숫자),
생성 버튼, 취소 버튼 (400*300)

방 만들기 생성 or 방 선택 후 입장 버튼 클릭하면 [게임 룸 panel]로 입장.

[게임 룸 panel]: 화면 중앙은 게임 보드판. 보드판 왼쪽엔 방장의 닉네임과 정보, 게임판 좌측엔 참여자의 
닉네임과 정보. 좌 하단은 유저 간 채팅, 우 하단 레디 버튼, 레디 취소 버튼 둘 다 레디 버튼 활성화 되면
5초 카운트 후 자동 게임 시작


게임은 더 지니어스 흑과백 룰에 기반한 단판, [게임 룸 panel] 좌 우측 각 유저 정보 밑에 세개의 원형 버튼이 있고 
한 판 이길 때 마다 불이 들어옴. 보드판 상단엔 진행중인 게임 세트 실시간 스코어 출력, 업데이트.
처음 게임 시작시에 게임 룰 설명 voice or text는 디폴트로 나옴. 유저 둘 다 skip버튼 클릭시엔 skip.
룰 설명 후에 가위바위보로 선공 후공 정함. 각 유저간 타일 제출 제한시간 1분, 10초부터 카운트다운 voice 출력. 
아무것도 내지 못하면 랜덤으로 제출. 한 라운드가 끝나면 직전 라운드에서 패배한 유저가 선공권을 가짐
